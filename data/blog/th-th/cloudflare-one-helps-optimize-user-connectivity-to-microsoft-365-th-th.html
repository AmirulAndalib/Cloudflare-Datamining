เรารู้สึกตื่นเต้นที่จะประกาศว่า Cloudflare ได้เข้าร่วมเป็นส่วนหนึ่งของ Microsoft 365 Networking Partner Program (NPP) เรียบร้อยแล้ว  โดย Cloudflare One ซึ่งมอบเส้นทางที่ปรับให้เหมาะสมกับการรับส่งข้อมูลจากลูกค้า Cloudflare ไปยัง Microsoft 365 เพิ่งผ่านการรับรองสำหรับ NPP ด้วยการแสดงให้เห็นว่า เส้นทานผ่านเครือข่ายของ Cloudflare ช่วยเพิ่มประสิทธิภาพในการเชื่อมต่อผู้ใช้เข้ากับ Microsoft ได้อย่างเหมาะสม

การเชื่อมต่อผู้ใช้เข้ากับอินเทอร์เน็ตบนเครือข่ายที่เร็วกว่า

ลูกค้าที่ปรับใช้ Cloudflare One ช่วยให้สมาชิกในทีมเข้าถึงเครือข่ายที่เร็วที่สุดในโลก โดยเฉลี่ย นี่คือเส้นทางไปยังอินเทอร์เน็ตส่วนที่เหลือ ผู้ใช้เชื่อมต่อจากอุปกรณ์หรือสำนักงานของตน และเข้าถึงเครือข่ายของ Cloudflare ในกว่า 250 เมืองทั่วโลก เครือข่ายของ Cloudflare ช่วยเร่งการรับส่งข้อมูลไปยังปลายทางสุดท้ายด้วยการผสมผสานระหว่างการกำหนดเส้นทางอัจฉริยะและการปรับปรุงซอฟต์แวร์

เรารู้สึกตื่นเต้นเช่นกันที่ในหลาย ๆ กรณี ปลายทางสุดท้ายที่ผู้ใช้ไปเยี่ยมชมอยู่ในเครือข่ายของ Cloudflare อยู่แล้ว Cloudflare ให้บริการคำขอ HTTP มากกว่า 28 ล้านรายการต่อวินาทีโดยเฉลี่ยแก่ลูกค้าหลายล้านรายที่รักษาความปลอดภัยแอปพลิเคชันของตนบนเครือข่ายของเรา เมื่อแอปพลิเคชันเหล่านั้นไม่ทำงานบนเครือข่ายของเรา เราสามารถพึ่งพาแกนหลักส่วนตัวระดับโลกของเราเองและการเชื่อมต่อของเรากับเครือข่ายมากกว่า 10,000 เครือข่ายทั่วโลกเพื่อเชื่อมต่อผู้ใช้

สำหรับการรับส่งข้อมูลของ Microsoft 365 เรามุ่งเน้นที่การแบ่งการรับส่งข้อมูลภายในเครื่องและกำหนดทิศทางให้มากที่สุดเพื่อนำผู้ใช้ไปยังเครื่องมือเพิ่มประสิทธิภาพการทำงานที่พวกเขาต้องการโดยไม่ทำให้ผู้ใช้ทำงานช้าลง โซลูชันการรักษาความปลอดภัยแบบเดิมสามารถแนะนำ hops หรือ backhauling เพิ่มเติมที่ทำให้การเชื่อมต่อกับเครื่องมือต่าง ๆ เช่น Microsoft 365 ช้าลง เราใช้ Cloudflare One เพื่อมอบความยืดหยุ่นในการระบุการรับส่งข้อมูลนั้นและให้เส้นทางที่ตรงที่สุดไปยังเครือข่ายปลายทางการบริการของ Microsoft ทั่วโลก

การรักษาความปลอดภัยข้อมูลและผู้ใช้ด้วย Cloudflare Zero Trust

ด้วยการตั้งค่านี้ การรับส่งข้อมูลที่เชื่อถือได้ไปยัง Microsoft จะใช้เส้นทางที่ตรงที่สุดโดยไม่ต้องดำเนินการเพิ่มเติม อย่างไรก็ตาม เราไม่ควรไว้วางใจอินเทอร์เน็ตส่วนที่เหลือ เครือข่ายของ Cloudflare ยังรักษาความปลอดภัยในการเชื่อมต่อ การสืบค้น และคำขอที่ทีมของคุณทำขึ้นเพื่อปกป้ององค์กรจากการโจมตีและการสูญหายของข้อมูล เราทำทั้งหมดนี้ได้โดยไม่ทำให้ผู้ใช้ช้าลงด้วยการมอบความปลอดภัยนั้นในศูนย์ข้อมูลภายในเซิร์ฟเวอร์ Edge ของเรา

แอปพลิเคชัน SaaS ที่จัดส่งทางอินเทอร์เน็ตสามารถทำให้อุปกรณ์ใด ๆ ที่มีเบราว์เซอร์เข้าสู่เวิร์กสเตชัน อย่างไรก็ตาม นั่นก็หมายความว่าอุปกรณ์เดียวกันเหล่านั้นสามารถเชื่อมต่อกับอินเทอร์เน็ตส่วนที่เหลือได้ ผู้โจมตีพยายามหลอกล่อผู้ใช้ให้เข้าสู่ไซต์ที่คล้ายคลึงกันเพื่อขโมยข้อมูลประจำตัว หรือพยายามให้ผู้ใช้ดาวน์โหลดมัลแวร์เพื่อลดทอนระบบป้องกันของอุปกรณ์ การโจมตีทั้งสองประเภทอาจทำให้ข้อมูลที่จัดเก็บไว้ในแอปพลิเคชัน SaaS ตกอยู่ในความเสี่ยงได้

Cloudflare ช่วยให้องค์กรหยุดการโจมตีประเภทเหล่านั้นผ่านกลยุทธ์การป้องกันในเชิงลึก โดย Cloudflare จะเริ่มต้นด้วยการส่งไฟร์วอลล์เครือข่ายรุ่นต่อไปในศูนย์ข้อมูลของเรา กรองการรับส่งข้อมูลสำหรับการเชื่อมต่อกับปลายทางที่อาจเป็นอันตราย หลังจากนั้น Cloudflare จะเรียกใช้ DNS Resolver ที่เร็วที่สุดในโลก และรวมเข้ากับข้อมูลเกี่ยวกับอินเทอร์เน็ตที่เหลือที่เราเห็น เพื่อกรองการสืบค้นไปยังโดเมนที่ทำฟิชชิงหรือไซต์ที่เป็นแม่ข่ายของมัลแวร์

สุดท้าย Secure Web Gateway ของ Cloudflare สามารถตรวจสอบการรับส่งข้อมูล HTTP เพื่อค้นหาการสูญหายของข้อมูล ไวรัส หรือสามารถเลือกแยกเบราว์เซอร์สำหรับไซต์เฉพาะหรือทั้งหมวดหมู่ได้ แม้ว่าเครือข่ายของ Cloudflare จะปกป้องผู้ใช้จากการโจมตีบนอินเทอร์เน็ตที่เหลือ แต่ Cloudflare One จะช่วยให้มั่นใจว่าผู้ใช้มีการเชื่อมต่อกับเครื่องมือ Microsoft 365 ที่พวกเขาต้องการได้โดยตรงและเป็นอิสระ

ด้วยการรับส่งข้อมูลที่ปลอดภัย Cloudflare ยังช่วยให้ผู้ดูแลระบบมองเห็นแอปพลิเคชันอื่น ๆ ที่ใช้ในองค์กรได้ หากไม่มีซอฟต์แวร์หรือคุณสมบัติเพิ่มเติม Cloudflare จะใช้ชุดความปลอดภัย Zero Trust เพื่อวิเคราะห์และจัดหมวดหมู่คำขอไปยังแอปพลิเคชันทั้งหมดในรายงาน Shadow IT ที่ครอบคลุม ผู้ดูแลระบบสามารถทำเครื่องหมายแอปพลิเคชันว่าได้รับการอนุมัติ ไม่ได้รับการอนุมัติ หรือไม่รู้จัก และอยู่ระหว่างการพิจารณา ตัวอย่างเช่น ผู้ดูแลระบบสามารถทำเครื่องหมายการรับส่งข้อมูล Microsoft 365 ว่าได้รับการอนุมัติ ซึ่งเป็นการตั้งค่าเริ่มต้นในการปรับใช้ที่ใช้การเปิดใช้งานแบบคลิกเดียวซึ่งเปิดตัวในวันนี้

ในบางกรณี ความสามารถในการมองเห็นได้นั้นก็ทำให้ประหลาดใจ ทีมความปลอดภัยและไอทีพบว่าผู้ใช้กำลังทำงานในแพลตฟอร์ม SaaS ที่ยังไม่ผ่านการตรวจสอบและอนุมัติจากองค์กร ในกรณีดังกล่าว ทีมสามารถใช้ Secure Web Gateway ของ Cloudflare เพื่อปิดกั้นคำขอไปยังปลายทางเหล่านั้น หรือแค่ป้องกันกิจกรรมบางประเภท เช่น การปิดกั้นการอัปโหลดไฟล์ไปยังเครื่องมืออื่นที่ไม่ใช่ OneDrive แต่ด้วย Shadow IT เราสามารถช่วยให้ทีมที่ใช้ Microsoft 365 มั่นใจได้ว่าข้อมูลจะอยู่ภายใน Microsoft 365 เท่านั้น

เรากับการเข้าร่วมใน Microsoft 365 Networking Partner Program

Cloudflare ได้เข้าร่วมเป็นส่วนหนึ่งของ Microsoft 365 Networking Partner Program (NPP) เรียบร้อยแล้ว โปรแกรมนี้ได้รับการออกแบบมาเพื่อให้ลูกค้าได้มีพันธมิตรกลุ่มหนึ่งซึ่งแนวทางปฏิบัติและคำแนะนำในการปรับใช้ของพันธมิตรนั้นสอดคล้องกับหลักการด้านเครือข่ายของ Microsoft สำหรับ Microsoft 365 เพื่อมอบประสบการณ์การใช้งานที่ดีที่สุดแก่ผู้ใช้ Microsoft ได้กำหนด NPP ขึ้นมาเพื่อให้ทำงานร่วมกับบริษัทเครือข่ายเพื่อการเชื่อมต่อที่เหมาะสมกับบริการของตน เรารู้สึกตื่นเต้นที่ได้ร่วมงานกับพันธมิตรที่มีเครือข่ายทั่วโลกและหลักการด้านความปลอดภัยที่สอดคล้องกับหลักการของเรา

ตั้งแต่วันนี้เป็นต้นไป เมื่อองค์กรที่ดำเนินการผ่าน Cloudflare One จะมีความสามารถในการตรวจสอบการเชื่อมต่อการรับส่งข้อมูล Microsoft 365 ได้โดยตรง ซึ่งช่วยให้ลูกค้าของเราที่มีไคลเอนต์ WARP ได้รับประโยชน์จากประสบการณ์ผู้ใช้ที่ราบรื่นสำหรับ Microsoft 365 ในขณะเดียวกันก็รักษาความปลอดภัยให้กับการรับส่งข้อมูลที่เหลือทั้งในแอป SaaS, แอปในองค์กร หรือการรับส่งข้อมูลทางอินเทอร์เน็ตโดยตรงผ่านชุดเครือข่ายและความปลอดภัยของ Cloudflare ทั่วโลกของผลิตภัณฑ์

ในการดำเนินการนี้ สิ่งที่ลูกค้าต้องทำคือเปิดใช้งานการตั้งค่าการเพิ่มประสิทธิภาพการรับส่งข้อมูล Microsoft 365 ในแดชบอร์ด Cloudflare One ของตน แม้ว่าการเชื่อมต่อ Microsoft 365 จะถูกกำหนดเส้นทางผ่านเกตเวย์ Cloudflare แต่เกตเวย์จะถูกจัดการโดยใช้การตั้งค่าเพิ่มเติมให้น้อยที่สุด เช่น นโยบาย "อย่าตรวจสอบ" จะถูกเปิดใช้งานโดยอัตโนมัติ

เปิดใช้งานได้ง่ายมากด้วยการคลิกเพียงไม่กี่ครั้ง:

  1. ล็อกเข้าสู่แดชบอร์ด Cloudflare for Teams
  2. ไปที่การตั้งค่า > เครือข่าย
  3. สำหรับการรับส่งข้อมูลแบบ Exclude Office 365 และการรับส่งข้อมูลแบบ Bypass Office 365 ให้คลิกสร้างรายการ
“เรารู้สึกตื่นเต้นที่ได้ต้อนรับ Cloudflare เข้าสู่ Networking Partner Program สำหรับ Microsoft 365” Scott Schnoll ผู้จัดการอาวุโสฝ่ายการตลาดผลิตภัณฑ์ของ Microsoft กล่าว “Cloudflare เป็นพันธมิตรที่ทรงคุณค่าซึ่งมุ่งเน้นให้ความช่วยเหลือลูกค้า Microsoft 365 นำหลักการเชื่อมต่อเครือข่าย Microsoft 365 ไปใช้ Microsoft แนะนำโซลูชันสำหรับการเชื่อมต่อกับ Microsoft 365 ให้กับสมาชิก Networking Partner Program เท่านั้น”

บทสรุป

องค์กรของคุณสามารถเริ่มปรับใช้ Cloudflare One ควบคู่ไปกับการใช้งาน Microsoft 365 ที่มีอยู่ได้แล้ววันนี้ เรารู้สึกตื่นเต้นที่จะได้ร่วมงานกับ Microsoft เพื่อให้สมาชิกในทีมของคุณสามารถเชื่อมต่อกับเครื่องมือที่จำเป็นสำหรับการทำงานของพวกเขาได้อย่างรวดเร็ว น่าเชื่อถือและปลอดภัย
