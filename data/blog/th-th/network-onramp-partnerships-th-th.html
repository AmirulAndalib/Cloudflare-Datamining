วันนี้เรารู้สึกตื่นเต้นที่จะได้ประกาศให้ทราบถึง พันธมิตรเครือข่าย On-ramp ล่าสุดของเราสำหรับ Cloudflare One Cloudflare One ถูกออกแบบมาเพื่อช่วยให้ลูกค้าเข้าถึงเครือข่ายทั่วโลกอย่างปลอดภัยและเหมาะสมที่สุด เราทราบถึงคำสัญญาในการเปลี่ยนลิงก์ MPLS กับเครือข่ายระดับโลกที่ปลอดภัย มีประสิทธิภาพและน่าจดจำจะเปลี่ยนแปลงเครือข่ายแบบองค์กร เพื่อตระหนักถึงวิสัยทัศน์ดังกล่าว เรากำลังจะเปิดตัวพันธมิตรซึ่งลูกค้าสามารถเชื่อมต่อไปยังเครือข่ายทั่วโลกของ Cloudflare’ จากอุปกรณ์ WAN & SD-WAN ที่เชื่อถือได้ที่มีอยู่แล้วและยังสามารถเชื่อต่อระหว่างกันแบบส่วนตัวผ่านศูนย์กลางข้อมูลที่พวกเขาตั้งอยู่ร่วมกัน

วันนี้เราจะเปิดตัวพันธมิตร WAN และ SD-WAN ร่วมกับ VMware, Aruba และ Infovista เรายังเพิ่ม Digital Realty, CoreSite, EdgeConneX, 365 Data Centers, BBIX, Teraco และ Netrality Data Centers ไปยังพันธมิตรเครือข่าย Interconnect เดิมของเราซึ่งประกอบด้วย Equinix ECX, Megaport, PacketFabric, PCCW ConsoleConnect และ Zayo อีกด้วย ปัจจุบันพันธมิตร Network On-ramp ของ Cloudflareได้ขยายผู้ให้บริการเชื่อมต่อชั้นนำ 15 รายใน 70 แห่งที่ไม่ซ้ำกัน ซึ่งทำให้ลูกค้าทำการการรับส่งข้อมูลบน Cloudflare ในวิธีที่ปลอดภัยและมีประสิทธิภาพไม่ว่าจะอยู่ที่ใดได้โดยง่าย

เชื่อมต่อไปยัง Cloudflare โดยใช้ผู้ให้บริการ WAN หรือ SD-WAN ที่คุณมีอยู่แล้ว

ด้วย Magic WAN ลูกค้าสามารถเชื่อมต่อกับศูนย์ข้อมูล สำนักงาน อุปกรณ์รวมถึงคุณสมบัติของคลาวด์บนเครือข่าย Cloudflare ได้อย่างปลอดภัยและยังสามารถกำหนดค่านโยบายเส้นทางเพื่อรับบิตไปยังที่ต้องการได้ทั้งหมดภายในโซลูชัน SaaS เพียงหนึ่งเดียว: ไม่จำเป็นต้องมีค่าใช้จ่าย MPLS หรือช่วงเวลานำการผลิตรวมถึงไม่ต้องมีบทลงโทษด้านประสิทธิภาพจากปัญหาการรับส่งข้อมูลที่เป็นทรอมโบนอีกต่อไป ในวันนี้หลายองค์กรใช้อุปกรณ์ SD-WAN แบบมีติดตั้งจริงหรือแบบเสมือนเพื่อกำหนดเส้นทางหรืออุโมงค์การรับส่งข้อมูลระหว่างสำนักงาน ศูนย์ข้อมูลและคลาวด์สาธารณะ นับจากวันนี้ลูกค้าเหล่านี้สามารถยกระดับการลงทุนในโครงสร้างพื้นฐานที่มีอยู่ก่อนหน้าได้ — ทั้งแบบกายภายพหรือแบบเสมือนบนคลาวด์ — เพื่อเชื่อมต่อการรับส่งข้อมูลไป Magic WAN โดยใช้คำสั่งง่ายๆ เพียงไม่กี่คำสั่ง

พิจารณาการตั้งค่าแบบง่ายด้านล่าง Magic WAN + พันธมิตรเครือข่าย Onramp อนุญาตให้คุณเชื่อมต่อกับระบบของตนเองและคลาวด์ VPC ในซับเนต RFC1918 โดยใช้อุปกรณ์ทั้งแบบติดตั้งจริงและเสมือนด้วยการเชื่อมต่ออย่างรวดเร็วและปลอดภัยโดย Magic WAN ยิ่งไปกว่านั้น Magic Firewall ยังอนุญาตให้คุณทำการกำหนดค่าและบังคับใช้กฎไฟล์วอลล์ที่ edge และผ่านไปยังการรับส่งข้อมูลที่ผjานไปทาง Magic WAN ได้อย่างต่อเนื่อง เครือข่ายอันกว้างขวางของ Cloudflare ที่แผ่ออกไปในกว่า 200 เมืองยังหมายถึงศูนย์ข้อมูลที่ใกล้ที่สุดอยู่ไม่ไกล

แม้ว่าการตั้งค่าที่เรียบง่ายและความเข้ากันได้นั้นยอดเยี่ยม แต่แผนงานที่ใช้ร่วมกันของเรากับพันธมิตรของเรานั้นมุ่งมั่นจะไปให้ไกลกว่านั้น วันนี้ การเชื่อมต่อเหล่านี้ถูกจัดทำผ่าน Anycast GRE และเราจะทำการประกาศให้ทราบถึงการสนับสนุน IPSec ที่จะเกิดขึ้นเร็วๆ นี้ซึ่งจะอนุญาตให้ลูกค้าสามารถเชื่อมต่อได้หลากหลายวิธี

ในการเปิดตัวในอนาคต การเข้าถึงอุปกรณ์พันธมิตรจะใช้เพียงข้อมูลอนุญาตเพื่อพิสูจน์ว่าพวกเขาเป็นส่วนหนึ่งของเครือข่ายลูกค้า Magic WAN และจะสามารถติดต่อไปยังการจัดการ API ของ Magic WAN รวมถึงกำหนดค่าด้วยตัวเองได้ — เป็นการตระหนักถึงวิสัยทัศน์ plug’n’play ของเครือข่ายคลาวด์อย่างแท้จริง นอกจากนี้ลูกค้า Magic WAN ยังสามารถใช้ประโยชน์จากการเพิ่มระดับความปลอดภัยด้วย Zero Trust ของ Cloudflare เพื่อการเข้าถึงแอปพลิเคชันและการใช้งานท่องอินเทอร์เน็ตที่เป็นการรวมการเข้าถึงเครือข่ายของ Zero Trust (ZTNA), Secure Web Gateway (SWG), Remote Browser Isolation (RBI), ความปลอดภัย DNS, ไฟล์วอลล์ L4 รวมถึงผลิตภัณฑ์อื่นๆ เมื่อมีการกำหนดให้เข้าสู่แพลตฟอร์มไร้รอยต่อเดียวกัน

" VMware SD-WAN จำลอง WAN เพื่อแยกบริการซอฟต์แวร์เครือข่ายออกจากฮาร์ดแวร์พื้นฐาน — ซึ่งสร้างความคล่องตัวและประสิทธิภาพสำหรับทุกกิจการและมันคือองค์ประกอบที่เป็นรากฐานของแพลตฟอร์ม VMware Secure Access Service Edge (SASE) VMware และ Cloudflare แบ่งปันวิสัยทัศน์เพื่อให้ลูกค้าได้ใช้งาน Global WAN อย่างคุ้มค่า พร้อมสรรพและปลอดภัยยิ่งขึ้น"
- Mark Vondemkamp รองประธานฝ่ายผลิตภัณฑ์, ธุรกิจ SD-WAN และ SASE, VMware
"Aruba ซึ่งเป็นบริษัทในเครือ Hewlett Packard Enterprise มีความยินดีที่จะร่วมงานกับ Cloudflare เพื่อพัฒนาโซลูชันที่จะทำให้ลูกค้าของเราสามารถใช้งานแพลตฟอร์ม Aruba EdgeConnect SD-WAN กับ Silver Peak ที่ได้รับมาตามการเชื่อมต่อระดับองค์กรบน Cloudflare Magic WAN และ Magic Firewall โซลูชันใหม่นี้สร้างบนแพลตฟอร์ม Aruba EdgeConnect ที่มีการบูรณาการในระดับเดียวกันที่ดีที่สุดกับการเชื่อมต่อกับผู้นำแห่งการเชื่อมต่อคลาวด์และการบริการด้านความปลอดภัยรวมถึงยังให้ลูกค้าสามารถใช้งานเครือข่าย Global Edge ของ Cloudfare เพื่อคุ้มครองและเร่งการทำงานของคลาวด์ได้"
- Fraser Street หัวหน้าพันธมิตรทางเทคนิค WAN สำหรับ Aruba

เชื่อมต่อโครงข่ายแบบส่วนตัวไปยัง Cloudflare ผ่าน PNI

การทำอุโมงค์การรับส่งข้อมูไปยังศูนย์ข้อมูล Cloudflare ที่ใกล้ที่สุด (ซึ่งอยู่ไม่เคยห่าง เนื่องจากเรามีศูนย์กระจายอยู่ในเมืองกว่า 200 เมือง) คือหัวใจของ Cloudflare Magic WAN เราใช้โปรโตคอลอินเทอร์เน็ตมาตรฐานเช่น GRE และ IPSec (จะมาเร็วๆ นี้) เพื่อส่งการรับส่งข้อมูลระหว่างเครือข่ายของเราและโครงสร้างพื้นฐานของลูกค้าอย่างปลอดภัย ในขณะที่การรักษาความปลอดภัยระดับโปรโตคอลนั้นยอดเยี่ยม เมื่อใดก็ตามที่คุณมีการเชื่อมต่ออินเทอร์เน็ตกับโครงสร้างพื้นฐาน นั่นหมายความว่าคุณเพิ่มพื้นที่การโจมตีให้กับการกำหนดค่าผิดพลาด

สำหรับองค์กรที่ต้องการระดับความปลอดภัยและประสิทธิภาพสูงสุดสามารถรวม Magic WAN เข้ากับ Cloudflare Network Interconnect Partnerสำหรับการใช้งานแบบส่วนตัว, เครือข่ายแบบ 2 เลเยอร์ที่ปลอดภัยและเชื่อถือได้ระหว่าง edge ของ Cloudflare และเครือข่ายของลูกค้า การเชื่อมต่อแบบส่วนตัวคืออีกหนึ่งอาวุธในการสร้างการป้องกันเชิงลึกสำหรับเครือข่ายของคุณ

เมื่อปีที่แล้วเรา ได้ประกาศ พัธมิตรการเชื่อมต่อโครงข่ายเครือข่ายรอบแรกของเรา ในวันนี้เราเพิ่ม Digital Realty, CoreSite, EdgeConneX, 365 Data Centers, BBIX, Teraco, และ Netrality Data Centers เพื่อให้บริการ:

  • Colocation คู่กับ Cloudflare ใน 70 แห่ง
  • ลดเวลานำการเชื่อมต่อข้ามเครือข่าย
  • สถาปัตยกรรมอ้างอิงการเชื่อมต่อ

ด้วยพันธมิตรใหม่ของเรา วันนี้ลูกค้ามีตัวเลือกมากขึ้นสำหรับการเชื่อมต่อที่เป็นส่วนตัวและปลอดภัยผ่านการแลกเปลี่ยนคลาวด์ (ซอฟต์แวร์ที่กำหนดคำสั่ง VLAN ผ่าน Dashboard) หรือด้วยการเชื่อมต่อแบบกายภาคที่เป็นส่วนตัว

“การรวมกันของ Cloudflare One และ PlatformDIGITAL® เปิดโอกาสให้ลูกค้าของเราเดินทางไปสู่การเปลี่ยนผ่านทางดิจิทัลได้เร็วยิ่งขึ้นและจัดการกับแรงโน้มถ่วงของข้อมูลได้โดยตรง คำแถลงการณ์ของเราได้วางแผนงานสำหรับการสร้างความสามารถดั้งเดิมแบบใหม่เพื่อรองรับการเชื่อมแพลตฟอร์มหลายแพลตฟอร์มโดยการร่วมมือกันในแต่ละอุตสาหกรรม การประกาศร่วมกับ Cloudflare ในวันนี้ถือเป็นก้าวที่สำคัญในการดำเนินตามวิสัยทัศน์นั้น โซลูชันของ Cloudflare เพื่อใช้ในการแก้ไขปัญหาเช่นการแปลข้อมูล การปฏิบัติตามข้อกำหนดและการรักษาความปลอดภัยสอดคล้องอย่างใกล้ชิดกับสถาปัตยกรรมศูนย์ข้อมูลที่แพร่หลาย PDx ของ Digital Realty™ วิธีการและความตั้งเสริมคุณค่าให้แก่ชุมชนข้อมูลที่มีการเชื่อมต่อมากมายบนแพลตฟอร์มระดับโลกของเรา”
- Chris Sharp ประธานเจ้าหน้าที่ฝ่ายเทคโนโลยี Digital Realty
“การร่วมงานของ CoreSite และ Cloudflare ทำให้ลูกค้าได้ใช้งานการเชื่อมต่อกับโดยตรงกับไฟเบอร์ความเร็วสูงรวมถึงได้รับความปลอดภัยที่เพิ่มขึ้นที่พวกเขาต้องการเพื่อเป็นไปตามข้อจำกัดที่เคร่งขัดรวมถึงการปฏิบัติตามข้อกำหนดที่สนับสนุนแอปพลิเคชันแบบไฮบริดที่ทันสมัย เรามีความตื่นเต้นที่จะเปิดการใช้งานบริการ Cloudflare Network Interconnect ภายในคลาวด์ที่มีเครือข่ายที่หนาแน่นของเราที่เปิดใช้งานในวิทยาเขตศูนย์ข้อมูลใน ลอส แองเจลิสและเดนเวอร์”
- Maile Kaiser SVP — ฝ่ายขายที่ Coresite
“เราตื่นเต้นที่จะแจ้งว่าพันธมิตรนี้ที่เกิดขึ้นจากวิสัยทัศน์ของ Cloudflare ในการ 'ช่วยสร้างอินเทอร์เน็ตที่ดีขึ้น' นั้นสะท้อนถึงปรัชญาของ BBIX ได้อย่างลึกซึ้ง ในสภาพแวดล้อมที่เต็มไปด้วยภัยคุกคามด้านความปลอดภัยของเครือข่ายที่เพิ่มขึ้น ลูกค้าของเราจำเป็นต้องพึ่งพาโซลูชันความปลอดภัยที่น่าเชื่อถือได้อย่าง Magic Transit เพื่อให้สามารถดำเนินธุรกิจต่อไปได้อย่างราบรื่น เราเชื่อว่าการเป็นพันธมิตรในครั้งนี้จะช่วยยกระดับคุณค่าของทุกบริษัทและช่วยให้พวกเราตอบสนอต่อความต้องการของตลาดที่เติบโตและหลากหลายมากขึ้นได้”
— Michikazu Fukuchi รองประธานบริหาร กรรมการและประธานฝ่ายปฏิบัติการของ BBIX Inc.

WAN ของอนาคตมาถึงแล้ววันนี้

ตลอด 10 ปีที่ผ่านมา Cloudflare ได้สร้างเครือข่ายที่เป็นหนึ่งในเครือข่ายที่เร็วที่สุด น่าเชื่อถือที่สุดและปลอดภัยที่สุดเครือข่ายหนึ่งของโลก สัปดาห์นี้เราได้ประกาศคุณสมบัติด้านประสิทธิภาพและความปลอดภัยเพิ่มเติมอีกหลายอย่างที่ให้ลูกค้าสามารถใช้ประโยขน์ด้านความปลอดภัยและความต้องการเครือข่ายได้ทั่วโลกได้เท่าเทียมกันมากขึ้น พันธมิตรเครือข่าย On-ramp ให้ตัวเลือกการบริการแบบส่วนตัว ปลอดภัยและประสิทธิภาพสูงเพื่อการเชื่อต่อแหล่งการรับส่งข้อมูลของคุณไปยัง Cloudflare

เพื่อร่วมเป็นส่วนหนึ่งของพันธมิตร On-ramp กรุณาติดต่อเราที่ onramppartners@cloudflare.com เราอยากฟังเรื่องเกี่ยวกับแพลตฟอร์มที่คุณใช้และที่อยากเห็นว่าเราร่วมเป็นส่วนหนึ่ง - ติดต่อเราได้ที่นี่

หากคุณต้องการเรียนรู้เพิ่มเติมเกี่ยวกับพันธมิตรเครือข่าย On-ramp, PNIs/CNIs แบบมีตัวตน, พันธมิตร/CNIs เสมือน รวมถึงวิธีที่พวกเขาควบรวมกับ Magic WAN ติดต่อทีมบัญชีของคุณหรือติดต่อไปที่ interconnection@cloudflare.com หากต้องการรายละเอียดเพิ่มเติมเกี่ยวกับ Magic WAN โปรดกรอกแบบฟอร์มติดต่อเราบน หน้าผลิตภัณฑ์
