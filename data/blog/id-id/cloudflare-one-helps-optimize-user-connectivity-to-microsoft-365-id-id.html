Kami sangat senang mengumumkan bahwa Cloudflare telah bergabung dengan Program Mitra Jaringan (Networking Partner Program, NPP) Microsoft 365.  Cloudflare One, yang menyediakan jalur yang dioptimalkan untuk traffic dari pelanggan Cloudflare ke Microsoft 365, baru-baru ini memenuhi syarat untuk NPP dengan menunjukkan bahwa on-ramp melalui jaringan Cloudflare membantu mengoptimalkan konektivitas pengguna ke Microsoft.

Menghubungkan pengguna ke Internet di jaringan yang lebih cepat

Pelanggan yang menggunakan Cloudflare One memberi anggota tim mereka akses ke jaringan tercepat di dunia, rata-rata, sebagai on-ramp mereka ke seluruh Internet. Pengguna terhubung dari perangkat atau kantor mereka dan mencapai jaringan Cloudflare di lebih dari 250 kota di seluruh dunia. Jaringan Cloudflare mempercepat traffic ke tujuan akhirnya melalui kombinasi perutean cerdas dan peningkatan perangkat lunak.

Kami juga senang bahwa, dalam banyak kasus, tujuan akhir yang dikunjungi pengguna sudah ada di jaringan Cloudflare. Cloudflare rata-rata melayani lebih dari 28 juta permintaan HTTP per detik, untuk jutaan pelanggan yang mengamankan aplikasi mereka di jaringan kami. Saat aplikasi tersebut tidak berjalan di jaringan kami, kami dapat mengandalkan tulang punggung pribadi global kami sendiri dan konektivitas kami dengan lebih dari 10.000 jaringan secara global untuk menghubungkan pengguna.

Untuk traffic Microsoft 365, kami berfokus membagi traffic secara lokal dan langsung mengantar pengguna ke alat produktivitas yang dibutuhkan tanpa memperlambat mereka. Solusi keamanan lama dapat mengakibatkan lompatan atau backhaul tambahan yang memperlambat konektivitas ke alat seperti Microsoft 365. Dengan Cloudflare One, kami memberikan fleksibilitas untuk mengidentifikasi traffic tersebut dan memberikan jalur paling cepat ke jaringan titik akhir layanan Microsoft di seluruh dunia.

Mengamankan data dan pengguna dengan Zero Trust Cloudflare

Dengan pengaturan ini, traffic tepercaya ke Microsoft menggunakan jalur paling langsung tanpa pemrosesan tambahan. Namun, bagian Internet lainnya tidak boleh dipercaya. Jaringan Cloudflare juga mengamankan koneksi, kueri, dan permintaan yang dibuat tim Anda untuk melindungi organisasi dari serangan dan kehilangan data. Kami dapat melakukannya tanpa memperlambat pengguna sebab kami memberikan keamanan tersebut di pusat data di edge kami.

Aplikasi SaaS yang dikirimkan melalui Internet dapat mengubah segala perangkat dengan browser menjadi workstation. Namun, itu juga berarti perangkat yang sama dapat terhubung ke seluruh Internet. Penyerang mencoba memancing pengguna ke situs yang mirip untuk mencuri kredensial, atau mereka mencoba meminta pengguna mengunduh malware agar dapat membobol perangkat. Jenis serangan apa pun dapat membahayakan data yang disimpan dalam aplikasi SaaS.

Cloudflare membantu organisasi menghentikan jenis serangan tersebut melalui strategi pertahanan mendalam. Pertama, Cloudflare memulai dengan memberikan firewall jaringan generasi terbaru di pusat data kami demi memfilter traffic untuk koneksi ke tujuan yang mungkin berbahaya. Selanjutnya, Cloudflare menjalankan resolver DNS tercepat di dunia dan menggabungkannya dengan data yang kita lihat di seluruh Internet untuk memfilter kueri ke domain phishing atau situs yang menghosting malware.

Terakhir, Secure Web Gateway Cloudflare dapat memeriksa traffic HTTP atas kehilangan data, virus, atau dapat memilih untuk mengisolasi browser untuk situs tertentu atau seluruh kategori. Selagi jaringan Cloudflare mengamankan pengguna dari serangan di seluruh Internet, Cloudflare One memastikan pengguna memiliki koneksi langsung dan tidak terbatas ke alat Microsoft 365 yang mereka butuhkan.

Dengan traffic yang diamankan, Cloudflare juga dapat memberi administrator visibilitas ke aplikasi lain yang digunakan di organisasi mereka. Tanpa perangkat lunak atau fitur tambahan apa pun, Cloudflare menggunakan rangkaian keamanan Zero Trust untuk menganalisis dan mengelompokkan permintaan ke semua aplikasi dalam laporan Shadow IT yang komprehensif. Administrator dapat menandai aplikasi sebagai disetujui, tidak disetujui, atau tidak diketahui, dan menunggu penyelidikan. Jadi sebagai contoh, Administrator dapat menandai traffic Microsoft 365 sebagai disetujui -- yang juga merupakan pengaturan default dalam penerapan yang menggunakan pengaktifan sekali klik yang dirilis hari ini.

Dalam beberapa kasus, visibilitas menghasilkan kejutan. Tim keamanan dan TI menemukan bahwa pengguna bekerja di platform SaaS yang belum ditinjau dan disetujui oleh organisasi. Dalam kasus tersebut, tim dapat menggunakan Secure Web Gateway Cloudflare untuk memblokir permintaan ke tujuan tersebut atau hanya mencegah jenis aktivitas tertentu, seperti memblokir unggahan file ke alat selain OneDrive. Dengan Shadow IT, kami dapat membantu tim yang menggunakan Microsoft 365 memastikan data hanya disimpan di Microsoft 365.

Partisipasi kami dalam Program Mitra Jaringan Microsoft 365

Cloudflare telah bergabung dengan Program Mitra Jaringan (Networking Partner Program, NPP) Microsoft 365. Program ini dirancang untuk menawarkan kepada pelanggan serangkaian mitra yang praktik dan panduan penerapannya selaras dengan prinsip jaringan Microsoft bagi Microsoft 365 guna memberikan pengalaman pengguna terbaik kepada pengguna. Microsoft mendirikan NPP untuk bekerja sama dengan perusahaan jaringan dalam upaya mengoptimalkan konektivitas terhadap layanannya. Kami bersemangat untuk bekerja sama dengan mitra yang jaringan global dan prinsip keamanannya sejalan dengan kami.

Mulai hari ini, melalui Cloudflare One, organisasi memiliki kemampuan untuk memastikan koneksi selangsung mungkin bagi traffic Microsoft 365. Ini memungkinkan pelanggan dan klien WARP kami diuntungkan dari pengalaman pengguna yang mulus untuk Microsoft 365, sekaligus mengamankan sisa traffic mereka ke aplikasi SaaS, aplikasi lokal, atau lalu lintas internet langsung melalui jaringan global Cloudflare dan rangkaian keamanan produk.

Untuk melakukan hal ini, semua pelanggan hanya perlu mengaktifkan pengaturan optimasi traffic Microsoft 365 di dasbor Cloudflare One mereka. Melalui pengaturan, sekalipun koneksi Microsoft 365 dirutekan melalui gateway Cloudflare, mereka ditangani dengan jumlah overhead tambahan paling sedikit, misalnya kebijakan "Jangan periksa" diaktifkan secara otomatis.

Sangat mudah untuk mengaktifkannya dengan beberapa klik saja:

  1. Login ke dasbor Cloudflare for Teams.
  2. Buka Pengaturan > Jaringan.
  3. Untuk traffic Exclude Office 365 dan traffic Bypass Office 365, klik Buat entri.
“Kami sangat senang menyambut Cloudflare ke dalam Program Mitra Jaringan untuk Microsoft 365,” ujar Scott Schnoll, Senior Product Marketing Manager, Microsoft. “Cloudflare adalah mitra berharga yang berfokus membantu pelanggan Microsoft 365 menerapkan Prinsip Konektivitas Jaringan Microsoft 365. Microsoft hanya merekomendasikan solusi anggota Program Mitra Jaringan untuk konektivitas ke Microsoft 365.”

Kesimpulan

Organisasi Anda dapat mulai menerapkan Cloudflare One hari ini bersama penggunaan Microsoft 365 yang ada. Kami senang bekerja sama dengan Microsoft untuk memberi anggota tim Anda konektivitas yang cepat, andal, dan aman ke alat yang mereka butuhkan untuk bekerja.
