在周日的 CIO Week 欢迎博客文章中，我们谈到 CIO 们为长期数字化转型寻找合作伙伴的价值。俗话说：“如果你想走得快，独自走，如果你想走得远，一起走。”

随着 Cloudflare 扩展到新的客户群体和新兴市场类别，例如 SASE 和 Zero Trust， 我们也越来越注重扩大与市场合作伙伴的关系(例如服务提供商、实施/咨询公司、系统集成商等)。由于安全和网络转型本身就令人生畏，因此在实施 Cloudflare One —— 我们的SASE零信任安全和网络服务平台时，客户通常需要战略顾问和实际支持。这些合作伙伴通过帮助客户评估、实施和管理我们的服务，在简化客户采用方面发挥了关键作用。

本文主要面向 Cloudflare 未来和当前的市场渠道合作伙伴，重点介绍我们在过去一年中如何发展合作伙伴计划，并将如何继续向前发展。

Cloudflare One：Cloudflare 合作伙伴中增长最快的产品组合

在过去的一年中，Cloudflare One 服务的采用是我们客户群中增长最快的领域。借助对渠道生态系统的投资，我们得以充分利用客户对 SASE 平台日益增长的需求，包括 Zero Trust 安全和云交付网络。

仅去年一年，Cloudflare One 的合作伙伴预订量就增长了 3 倍。与此同时，成交合作伙伴的数量同比增长了 70%。

合作伙伴反复提到，我们的平台部署和管理非常简单，我们的创新步伐让他们对我们的路线图充满信心，我们的全球网络确保了规模、速度和韧性，这些都是推动客户对 Cloudflare One 服务强烈需求的关键因素。

从传统的本地设备迁移到云交付的 SASE 架构是一个过程。对于大多数客户来说，合作伙伴帮助将这一过程分为两大类，大致的定义是：网络层转型和 Zero Trust 安全现代化。

网络层转型

多云和混合云架构日益成为常态。随着企业采用这种方法，它们的网络基础设施同样需要适应，以便能够轻松连接到各种云环境。

传统上依赖 SD-WAN 和 MPLS 技术的组织将转向基于云的网络即服务 (NaaS)产品，例如 Cloudflare 的 Magic WAN (Cloudflare One 平台的一部分) 以增加灵活性并降低成本。这也将为新一代云网络专家和顾问带来收入机会，他们具备能力帮助组织从传统的本地硬件迁移到 NaaS 架构。

对于一些组织来说，与开始 Zero Trust 安全迁移相比，网络转型实际上可能是一个更有吸引力的初始入口点，因为 NaaS 允许组织维持现有的安全工具，同时仍然提供一条战略路径，以迈向未来具有云交付保护的完整无边界架构。

实施 Zero Trust 架构

对于今天的许多组织来说，以 Zero Trust 的最佳实践实现员工、设备、数据和办公室的安全现代化，是同样重要的优先事项。混合办公和远程工作的趋势给 IT 和安全团队带来了额外的压力，他们需要重新考虑如何保护对公司资源的访问，并摆脱传统的 ‘城堡加护城河’ 架构。Zero Trust 承诺提供更强的可见性和更精细化的控制，以及覆盖所有流量的身份感知保护，而不管其来源或目的地为何。

虽然迁移到 Zero Trust 架构的好处不容否认，但实现完整 Zero Trust 架构的过程通常需要第三方的帮助。根据 iVanti 最近发布的一份报告，73% 的公司计划在未来 18 个月迁移到基于云的架构，其中 46% 的公司 IT 安全团队对自己应用 Zero Trust 模型的能力缺乏信心，这就是为什么有 34% 的公司依赖第三方安全提供商来帮助他们实现零信任。1 这是合作伙伴可以提供帮助的地方。

推出 Cloudflare One 授权服务交付合作伙伴认证计划

Cloudflare 高度专注于构建市场上最令人信服、易于使用的 SASE 平台，以帮助企业加快网络和安全架构的转型。我们的全球网络覆盖 100 多个国家/地区的 275 多个城市，拥有 172 Tbps 的网络容量，其规模和韧性确保我们能够可靠、高速地提供保护，无论客户位于世界任何地方。

随着我们的数据中心物理网络不断扩展，我们的战略渠道合作伙伴网络也在不断扩大，我们依赖其为客户提供专业的管理服务，客户可能需要这些服务作为其 Cloudflare One 部署的一部分。Cloudflare 正在与全球各地合作伙伴积极合作，以打造咨询、迁移和管理服务，旨在以合作伙伴服务专业知识支持 Cloudflare One 实施，以确保 100% 的客户采用和满意度。

为了帮助合作伙伴发展他们的 Cloudflare One 服务专业知识，并在市场上脱颖而出，我们今天隆重宣布在有限范围内推出全新的授权服务交付合作伙伴认证计划。这个项目旨在向满足 Cloudflare 有关提供 Cloudflare One 相关专业服务的高标准的合作伙伴提供授权。

要成为授权合作伙伴，合作伙伴需要经过严格的技术验证过程，其服务交付能力的安全性、性能和可靠性指标也将经过评估。获得授权服务合作伙伴称号的合作伙伴将享有各种好处，例如：

  • 参与需要 Cloudflare One 服务的商机
  • 指定 Cloudflare One 合作伙伴服务交付经理协助打造其服务实践
  • 获得特别合作伙伴激励基金，其旨在确保授权合作伙伴服务被积极用于 Cloudflare One 客户互动。

为了支持这一全新合作伙伴认证计划，我们同时推出高级启动和培训路径，包括讲师培训和通过合作伙伴门户网站提供的在线课程，以及先进实验室环境，旨在帮助合作伙伴学习如何实施和支持 Cloudflare One 部署。成功达到 ADSP 要求的合作伙伴还将获得观摩客户部署的机会，以进一步提高其能力和专业知识。

对于对此计划感兴趣的现有和潜在 Cloudflare 合作伙伴，我们将推出新的 Cloudflare 授权服务交付合作伙伴认证清单，其中包括有关申请流程的详细信息。

如果是 Cloudflare 现有合作伙伴，您还可以联系指定的渠道客户经理以了解更多信息。
