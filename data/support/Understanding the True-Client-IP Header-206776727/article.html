Learn how to see end-user IP addresses in origin requests with Cloudflare's True-Client-IP Header.

Overview

Connections from Cloudflare to origin servers come from Cloudflare IPs. True-Client-IP is a solution that allows Cloudflare users to see the end user’s IP address, even when the traffic to the origin is sent directly from Cloudflare.

For more details on what True-Client-IP is, refer to our product documentation.

Enable True-Client-IP Header

To add the True-Client-IP header to incoming requests, enable the Add “True-Client-IP” header Managed Transform.

For more information, refer to Managed Transforms in the developers documentation.

Note: Before Managed Transforms were available, you added the True-Client-IP header by enabling True-Client-IP Header in the Network app of the Cloudflare dashboard. The current recommended procedure is to enable the Managed Transform.
